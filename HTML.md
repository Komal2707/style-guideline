## HTML Style Guide



# Markup structure

```html
<div class="fake-class">
    <div class="child-class">
        Text or other elements. always 1 line break between siblings.
    </div>
    
    <div class="child-class">
        No line break between parent and first child/ parent and last child
    </div>
</div>

<label>Label always on same line</label>

<div
     class="some-class"
     other-attr="someValue"
     extra-attr="someOtherValues"
>
    Alway have the attributes on same tab spaces to make it look clean and visible
</div>
```
